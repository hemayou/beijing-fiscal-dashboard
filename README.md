# 京财月读 · 北京财政景气月度仪表盘

北京市全市一般公共预算月度收支（2022.01–2026.05，年初至今累计口径）的动态分析专题页。
深/浅双主题，ECharts 动态图表：收支双轨、月度节奏、同比景气、税基结构、支出领域、执行进度。

- 在线访问：见仓库 Settings → Pages 生成的链接
- 数据来源：北京市财政局·财政收支情况 https://czj.beijing.gov.cn/zwxx/czsj/czsz/
- 口径：**全市**（市区两级合计）·仅**一般公共预算**·均为**累计值**（详见页脚说明）

纯静态单页：`index.html` + `data.js`，无需后端。图表库 ECharts 与字体经 CDN 加载。
